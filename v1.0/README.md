# Chess AI - Polytech Marseille
Ce projet consiste à créer un jeu d'echec en C.
Le joueur se bat contre une IA.

## features :
- algorithme d'analyse du meilleurs coup avec profondeur prédéfini.
- fichier des ouverture et fermeture, pour avoir les meilleurs coups sans calculs.
- interface graphique
- hebergement sur un serveur et frontend site web si possible.
